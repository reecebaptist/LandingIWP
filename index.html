About The Project

About The Project

Convention planner is a flexible event organising tool. It helps to create and schedule events more easily. It has features like predicting budget and the time needed to start the backgrond work. We are also trying analyse the events data and show the profits. And it also shows recommendations based on users history of registered events.

F.A.Qs

  • What does it do?

    The project developed assists in conducting new events in one convention, helps to track participant information, budget information, transaction details, etc.

  • Why is it needed?

    In the recent era, the number of conventions taking place are increasing. They offer a whole range of events which a lot of people from many different fields participate.

  • What are the new features?

    It can also notify participants via emails. It also has a budget prediction feature based on a Machine Learning Algorithm.
